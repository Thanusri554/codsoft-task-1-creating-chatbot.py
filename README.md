# ChatBot

This repository contains a simple command-line ChatBot implemented in Python (chatbot.py).

Quick start

1. Clone the repo or download chatbot.py:
   git clone https://github.com/Thanusri554/codsoft-task-1-creating-chatbot.py.git
2. Run the bot:
   python chatbot.py

Description

The bot recognizes basic inputs: "hi"/"hello", "how are you", "what is your name", and "bye" to exit.
